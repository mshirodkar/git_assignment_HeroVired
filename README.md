# CalculatorPlus 

A simple Python calculator supporting basic arithmetic operations and square root functionality.

##  Features

- Addition
- Subtraction
- Multiplication
- Division (with zero division error handling)
- Square root (with negative number check)

##  Branch Structure

- `main`: Production-ready code
- `dev`: Active development branch
- `feature/sqrt`: Feature branch for square root functionality

##  Releases

- `v1.0`: Basic operations
- `v2.0`: Added square root & fixed divide by zero bug

python calculator.py
